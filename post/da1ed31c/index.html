GitHub 脚本 & 扩展

脚本

GitHub 菜单：为 GitHub 添加更多菜单项，快速跳转到需要的页面

Github 增强 - 高速下载：为 GitHub 文件添加高速下载按钮，解决下载速度慢的问题

FastGithub 镜像加速访问、克隆和下载：另一个加速访问、下载的脚本，启用一个就行

显示 Github1s 按钮：为GitHub仓库添加快速跳转到GitHub1s的按钮

插件

Little Star - Github Stars Manager：一个用于管理Github Star项目的Chrome拓展

Git Master：Git文件树的Git主扩展，支持GitHub、GitLab、Gitee、Gitea

Octotree - GitHub code tree：与Git Master类似的工具

GitHub Web IDE：在线Web IDE中打开GitHub存储库

Enhanced GitHub：浏览器扩展以显示每个文件的大小，下载链接并直接将文件内容复制到剪贴板
